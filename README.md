# blog-entry
merge posts to avant-gardes.github.io

## 서버올리기

```bash
npm start
```


## 글 작성하기

제목은 영어 스네이크 케이스로 작성한다 (화면에 표시하는 타이틀은 본문에서 한글로 바꿀 수 있음)

```bash
hexo new post my-new-blog-title
```

## 작성한 글 올리기
github.com/avant-gardes/avant-gardes.github.io 로 포스팅한다.
```bash
npm run deploy
```
